# ICSI 418Y Homework 1 - se-login-signup
The goal of this project is to gain hands-on experience of working with web application development using HTML and
CSS and version control.

## Project Structure:
 - `login.html`: page containing fields to input a user ID and password, a submit button, and a link to the signup page
 - `signup.html`: page containing fields to input a first name, last name, user ID and password, and a submit button and link to the login page
 - `styles.css`: style sheet used for formatting
